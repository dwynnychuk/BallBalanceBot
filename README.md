# BallBalanceBot
## Summary
A 3-DOF robotic project that balances a ball in the center of a platform. 

## Structure
.
├── main.py
├── cameraCalibration.py
├── classCamera.py
├── classControl.py
├── classRobot.py
├── classServo.py
└── logger.py

## Usage
Without camera display 
`python3 main.py`

With camera display
`python3 main.py --display`
